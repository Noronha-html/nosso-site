O que é o front-end?

Front-end, ou desenvolvimento de interface de usuário, é uma parte essencial do design de sites e aplicativos. Ele se concentra na criação de experiências interativas e visuais para os usuários, lidando com tudo que você vê e interage em uma página da web. O front-end é toda a interface gráfica de um projeto, onde é desenvolvida a aplicação que o usuário terá interação direta. A pessoa desenvolvedora front-end cria sites e aplicativos usando linguagens de programação da web, tais como o HTML, CSS e JavaScript.

Linguagens

HTML CSS e JavaScript

O HTML (Hypertext Markup Language) é uma linguagem de marcação que estrutura conteúdos em páginas web, o CSS (Cascading Style Sheets) é uma linguagem de estilos que define o design e layout das páginas, já o JavaScript é uma linguagem de programação que adiciona interatividade e dinamismo às páginas web.

  • Java

Ferramenta

Visual Studio Code

Conhecido popularmente como VS Code, é um editor de código-fonte desenvolvido pela Microsoft. Ele se destaca por ser leve, rápido e altamente personalizável, o que o torna uma das ferramentas favoritas entre desenvolvedores de diversas linguagens de programação.
